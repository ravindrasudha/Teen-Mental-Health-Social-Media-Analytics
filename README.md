# 📊 Teen Mental Health & Social Media Analytics

## Project Overview
This project analyzes the relationship between social media usage and mental health among 1,200 teenagers aged 13–19.

## Tools Used
- Power BI
- Excel
- DAX
- Data Visualization

## Key KPIs
- Total Participants: 1,200
- Average Social Media Usage: 4.5 Hours
- Average Sleep: 6.5 Hours
- Depression Rate: 2.6%
- Average Stress Level: 5.5/10
- Average Anxiety Level: 5.6/10

## Key Insights
### Heavy Social Media Usage
Users spending 6+ hours daily show significantly higher depression rates.

### Sleep Crisis
Average sleep duration is below the recommended 8–10 hours.

### Platform Analysis
TikTok users showed slightly higher depression rates compared to Instagram users.

### Mental Health Correlation
Stress, anxiety, and social media usage show positive correlations with depression.
